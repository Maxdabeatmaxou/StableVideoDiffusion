# Stable Video Diffusion - FULL STACK

Contenu :
- Dockerfile complet (SVD + Gradio + FastAPI)
- Téléchargement automatique du modèle
- Interface Web UI (Gradio)
- API REST `/video` via FastAPI
- Compatible RunPod (port 7860)

Déploiement :
1. Dockerfile path : `Dockerfile`
2. Port exposé : 7860
3. Utiliser Web UI ou appeler l'API POST `/video`
